TechVault
TechVault is a modern, fast, and secure inventory management web application. It is built to help users easily track, organize, and manage their tech gadget collection in one place.

Live Links
Live Website: https://techvault-by-sayed.vercel.app/

Purpose of the Project
The main goal of TechVault is to simplify inventory management for tech enthusiasts. It provides a clean and user-friendly interface to manage products, monitor stock levels, and categorize tech gear, making it easy to keep track of every gadget you own.

Key Features
Secure Authentication: User-friendly login and registration system powered by Firebase.

Full CRUD Operations: Effortlessly add, view, update, and delete gadgets from your inventory.

Responsive UI: A modern, mobile-first design built with Tailwind CSS that looks great on any device.

Category Filtering: Quickly find specific gadgets by filtering them based on categories.

Real-time Management: Seamlessly manage product details and keep your dashboard up-to-date.

Quick Setup & Installation
Follow these steps to run the project on your local machine:

Clone & Install:

Bash
git clone <your-repository-url>
cd <your-folder-name>
npm install
Configure: Create a .env.local file in the root directory and add your Firebase credentials.

Run:

Bash
npm run dev
Technologies Used
Framework: Next.js (for high performance and routing)

Styling: Tailwind CSS (for modern and responsive design)

Database & Auth: Firebase (for secure data storage and user management)

Language: JavaScript/React
