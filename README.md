Description---

TechVault is a modern inventory management web application built with Next.js, Tailwind CSS, and Firebase. It provides a secure and efficient way to manage tech gadgets, track stock levels, and organize products by category.



Key Features---

Secure Auth: Firebase-powered login and registration.
CRUD Operations: Easily add, view, and manage gadget inventory.
Responsive UI: Modern, mobile-first design using Tailwind CSS.
Category Filtering: Organized views for different types of tech gear.



Quick SetupClone & Install---

1.Clone & Install:
    git clone <repo-url>
    cd <repo-name>
    npm install
2.Configure: Create a .env.local file and add your Firebase credentials.
3.Run:
    npm run dev
