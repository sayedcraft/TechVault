# 🚀 TechVault

**TechVault** is a modern, fast, and secure inventory management web application. It is designed to help tech enthusiasts track, organize, and manage their gadget collections seamlessly in one place.

---

## 🌐 Live Demo
🔗 **[Visit TechVault](https://techvault-by-sayed.vercel.app/)**

---

## 🎯 Purpose of the Project
The primary goal of **TechVault** is to simplify gadget management. It provides a clean, user-friendly interface to:
* Monitor stock levels in real-time.
* Organize tech gear by category.
* Keep a comprehensive record of all owned gadgets.

---

## ✨ Key Features
* **🔐 Secure Authentication:** Robust login and registration system powered by **Firebase**.
* **🛠️ Full CRUD Operations:** Effortlessly add, view, update, and delete gadgets from your inventory.
* **📱 Responsive UI:** Modern, mobile-first design built with **Tailwind CSS**, ensuring a great experience on any device.
* **🔍 Category Filtering:** Quickly find specific gear by filtering products based on their categories.
* **⚡ Real-time Updates:** Seamlessly manage product details and keep your dashboard synchronized.

---

## 🛠️ Technologies Used
* **Framework:** [Next.js](https://nextjs.org/) (for high performance and routing)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/) (for modern, responsive design)
* **Database & Auth:** [Firebase](https://firebase.google.com/) (for secure data and user management)
* **Language:** JavaScript, React
